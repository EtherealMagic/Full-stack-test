# Full-stack-test
Mi primer proyecto en desarrollo web para probar las distintas etapas de un desarrollo full stack.

[Link de la página](https://tobiasivangodoy.github.io/Full-stack-test/) (la primera llamada al backend puede tardar porque se queda dormido).

## Pasos
1) Crear el HTML
2) Crear el CSS
3) Crear el JavaScript del frontend
4) Crear el backend en Python + SQLite
5) Crear las APIs REST
6) Conectar el front y back
7) Testear casos

## Requisitos
- En el documento backend/requirementes.txt

## Modo de uso (local)
1) Descargar el repositorio en .zip y descomprimirlo.
2) Cambiar la ruta de conexión en "script.js".
2) Correr por consola "python app.py" y mantener abierto.
3) Abrir "index.html" para ver la pagina (En VS code corre sin pasos extra).
